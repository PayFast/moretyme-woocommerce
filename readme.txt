PayFast MoreTyme Module for WooCommerce
----------------------------------------------------------------------------------------------------------

INTEGRATION:
1. Unzip the module to a temporary location on your computer.
2. Copy the “wp-content” folder in the archive to your base “wordpress” folder
- This should NOT overwrite any existing files or folders and merely supplement them with the PayFast files.
- This is however, dependent on the FTP program you use.
3. Login to the WordPress Administrator console
4. Using the main menu, navigate to Plugins.
5. Click “Activate” on “PayFast MoreTyme”

REQUIREMENTS:
MoreTyme needs to be activated on your PayFast account https://www.payfast.co.za

Note: A PayFast MoreTyme banner will be displayed on all products over R50
